## ✨ Features  
- 🌦️ **Real-time Weather**: Fetch current weather for any city.  
- 📍 **Location Search**: Auto-complete and geolocation support.  
- 📊 **5-Day Forecast**: Visualize temperature trends.  
- 🌙 **Dark/Light Mode**: Toggle for user preference.
